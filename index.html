About


Newman API allows you to add a contact form to your (static) website. Just add some HTML code to your site and you're on your way. No programming is required. Not even javascript.

  
<!DOCTYPE html>
<html lang="en">
  <head>
    <!-- name your masterpiece -->
    <title></title>
  </head>
  <body>
    <div id="artist">
      <!-- some optional info -->
      by <strong>YOUR NAME</strong><br />
      <a href="http://www.example.com" target="_blank">http://www.example.com</a>
      <hr />
      <br /><br />
    </div>
    <div id="artwork"></div>
    <script type="text/javascript">
      var artwork = '';

      // YOUR CODE HERE

      document.getElementById('artwork').innerHTML = artwork;
    </script>
  </body>
</html>
